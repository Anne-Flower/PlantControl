# Plant Control

### Features :
- add a plant : name, photo
- add an intervall to schedule the watering.
- add a notification, 1 day before to prevent and the day of the watering.
- responsive
- accessibility

### Techno:
- pwa
- React, TS, Tailwind : Front
- Node : Back
- Firebase : stockage cloud / auth
- Optionnal : tests playwright



In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
